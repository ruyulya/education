# Help commandline

## How to open terminal
1. Find app terminal
2. Write cd and add way to folder

## How to change directory

### Windows
```sh
cd C:\folder_name
```
### MacOS/Linux
```sh
cd/folder_name
```

## Listing directory
### Windows
```sh
dir
```
### MacOS/Linux
```sh
ls
```
 ## How to leave VIM
 ### Without save
```sh
Esc
:q!
```
### With save
```sh
Esc
:wq
